# HypeScope Bot

A Discord bot built with py-cord.

## Setup

1. Install dependencies:

   ```
   pip install -r requirements.txt
   ```

2. Copy `.env.example` to `.env` and add your Discord bot token:

   ```
   cp .env.example .env
   ```

3. Run the bot:
   ```
   python bot.py
   ```

## Commands

- `/hello` - Say hello
- `/info` - Get bot information
